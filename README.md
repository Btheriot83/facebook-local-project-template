# Facebook Local Project Template

Reusable Facebook operating system for local service businesses: lightweight execution template + full strategic playbook, packaged as a Claude/OpenClaw skill bundle.

## What's in this bundle

- `SKILL.md`:
  Primary operator template for running local Facebook content and campaign execution in a repeatable weekly system.
- `references/`:
  Supporting docs used by the primary template.
- `playbook/SKILL.md`:
  Full reference playbook with deeper strategy, offer mapping, audience handling, and messaging frameworks.
- `playbook/references/`:
  Supporting docs for the full playbook.

## Who this is for

This bundle is designed for local, trust-driven businesses that win by showing real work, real people, and real outcomes:

- Diesel repair and fleet service
- Contractors and trades
- Medspas and aesthetics clinics
- Healthcare and specialty clinics
- Home services (HVAC, plumbing, electrical, cleaning, roofing)
- Community-heavy local brands that rely on reputation and repeat business

## Why two skills

- Use the root `SKILL.md` when you want fast execution with minimal overhead.
- Use `playbook/SKILL.md` when you need full planning depth, positioning clarity, or campaign resets.

This gives teams a practical default mode plus a deeper reference mode without forcing complexity into daily execution.

## LarryBrain install flow

1. Open LarryBrain Marketplace publishing/import flow.
2. Use this repository URL as the source.
3. Confirm the entry skill is root `SKILL.md`.
4. Publish, then keep `playbook/` available as the advanced reference layer.

For official publishing details, see:
- https://www.larrybrain.com/docs/publishing
